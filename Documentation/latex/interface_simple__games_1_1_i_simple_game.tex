\subsection{Detailed Description}
Interface created for game windows. 

Each class implementing \doxyref{I\+Simple\+Game}{p.}{interface_simple__games_1_1_i_simple_game} have to provide {\ttfamily Init()} and {\ttfamily Reset()} methods. Their task is to initialize game window accordingly at the beginning of the game and after restart.
