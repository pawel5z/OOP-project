Detailed Description

This is lazy container i.e. it doesn't contain all phrases in one of its fields. Instead of that, it calculates number of available phrases upon construction and then, when asked for random phrase, it uses App's static rng to generate random number between 0 and count and reads from assets\phrases.dat till that number.
